Pi 拍錢包綁定 Pi 信用卡/ Pi 拍兔Debit卡筆筆回饋 5%無上限

  • 活動時間：2020/3/1 00:00-2020/8/31 23:59
  • 活動內容：活動期間於PChome 24h購物選擇 Pi 拍錢包綁定 Pi 信用卡/ Pi 拍兔Debit卡付款筆筆享5% P幣回饋無上限。
  • 付款方式：選擇Pi 拍錢包綁定 Pi 信用卡全額支付結帳享以上優惠
  • 注意事項：
    • Pi 信用卡之正卡、附卡持卡人均可參加 P幣回饋計劃(正、附卡 P幣分開計入至各自Pi拍錢包帳戶)。
    • Pi 信用卡簽帳消費將不再享有本行紅利點數、現金回饋等活動。
    • 使用 Pi 信用卡國內新增一般消費可累積 P幣，回饋之 P幣未足1點採四捨五入計算(例如：國內ㄧ般刷卡消費40元x2%=0.8點，四捨五入為1點)。
    • P幣回饋將於刷卡後2~3個工作日內(依廠商請款時間為準)回饋至持卡人 Pi 拍錢包帳戶(持卡人註冊 Pi 拍錢包之身分證需與本行留存一致，且 Pi 信用卡須綁定至 Pi 拍錢包始享 P幣回饋)。
    • 若因 Pi 信用卡補換發且卡號未異動，則無需於 Pi 拍錢包重新綁定 Pi 信用卡；若其它異動卡號情況，請再重新綁定 Pi 信用卡
    • 若持卡人同時擁有兩張 Pi 信用卡，可同時綁定於 Pi 拍錢包中享有 P幣回饋。
    • 於 Pi 拍錢包匯入 P幣之日起計180天內需使用完畢(非自申請 Pi 拍錢包帳戶日期起)，超過使用期限 P幣將自動歸零失效，不得要求兌換成現金、紅利點數等其他權益。
    • 持卡人若因任何原因取消交易/退貨、刷卡爭議退還款項或帳務調整等之情事，將依其退還款項金額計算應扣回之 P幣，並於持卡人 Pi 拍錢包帳戶扣除 P幣；如持卡人 Pi 拍錢包帳戶之P幣不足或已全數兌換完畢，本行得以1 P幣換算為新臺幣1元後，列示為信用卡帳單之應繳金額向持卡人收取。
